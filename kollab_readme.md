---

### 2. Install dependencies
```bash
npm install
```

---

### 3. Environment Variables

Create a `.env` file in the root of the project and add:

```env
PORT=9000
MONGO_URI=your mongo url
NODE_ENV=development
```

---

### 4. Run the server

```bash
npm run dev
```

The API will start on:

```
http://localhost:9000
```
